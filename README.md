README – Cadastro do Altabus

Descrição:
Formulário HTML para cadastro de novos usuários no sistema Altabus.

Campos:

Nome Completo
CPF
Data de Nascimento
E-mail
Telefone
Usuário
Senha
Confirmar Senha

Tecnologia:

HTML5

Objetivo:
Permitir o cadastro de usuários e praticar a criação de formulários em HTML.
